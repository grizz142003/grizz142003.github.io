X86 Architecture Operating System

Printf functionality

Currently developing a basic operating system that runs within the Bochs x86 emulator. Implemented a custom framebuffer driver with support for screen output features such as line breaks and clear screen functionality, designed to emulate the behavior of C’s printf function from the standard I/O library.
